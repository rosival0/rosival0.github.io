1. Conceitos
    1. Conceito de um Sistema Operacional
      • Formado por um ou mais softwares.
      • Software que garante a interface entre o computador e o usuário.
    2. Conceito de Hardware
      • Se trata da parte física de um computador.
      • Processamento de dados através da formação de dispositivos.
    3. Conceito de Software
      • Se trata de todo programa presente em qualquer dispositivo.
      • Um serviço computacional.
  2. Sobre os Sistemas Operacionais
    1. Quais os tipos de sistemas operacionais?
      • Windows
      • Linux
    2. Qual a função de um sistema operacional?
      • Gerenciar arquivos.
      • Executar programas.
    3. Onde o sistema operacional é armazenado?
      • Disco rígido do computador.
      • Disco interno ou externo.
  3. Sobre o Hardware
    1. Qual a função do hardware?
      • Se dá pela montagem de equipamentos.
      • Fazer o equipamento funcionar após a montagem.
    2. Tipos de hardware.
      • Hardware de armazenamento.
      • Hardware de processamento.
      • Hardware de entrada.
      • Hardware de saída.
    3. Cite respectivamente um hardware de cada tipo.
      • HD.
      • Processador.
      • Mouse.
      • Caixas de som.
  4. Sobre o Software
    1. Qual a função de um software?
      • Faz a máquina compreender comandos do usuário.
      • Executa os comandos do usuário.
    2. São tipos de software.
      • Software de programação.
      • Software de aplicação.
      • Software de sistema.
    3. A importância de um software.
      • Transforma dados em informações.
      • Responsável pela lógica de um sistema.

    1. Quais os componentes de um sistema?
      • Recursos de dados.
